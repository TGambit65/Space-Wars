# Progression

Space Wars 3000 features a leveling system, skill trees, and a technology tree that unlock new abilities and improvements as you play.

## Leveling

Your character earns XP from combat, trading, and missions. Each level grants **1 skill point**.

## Skills

Spend skill points on 8 skills. Each skill has a maximum level of **10**.

| Skill | Effect per Level | Description |
|-------|-----------------|-------------|
| Combat Mastery | +3% weapon damage | Increases all weapon damage output |
| Shield Expertise | +4% shield capacity | Boosts total shield hit points |
| Trade Acumen | +2% trade profit | Better buy/sell prices at ports |
| Navigation | -3% fuel usage | Reduces fuel consumed per jump |
| Engineering | +5% repair effectiveness | Cheaper and better repairs |
| Leadership | +2% crew effectiveness | All crew bonuses improved |
| Mining Proficiency | +5% resource extraction | More resources from colonies |
| Scanner Mastery | +4% scan range | Extended scanning distance |

> **Tip**: Early on, invest in Navigation and Trade Acumen to maximize profits. Shift to Combat Mastery and Shield Expertise once you're ready for tougher fights.

## Technology Tree

Research technologies to unlock new capabilities. Technologies cost credits and take real time to complete.

| Technology | Prerequisites | Cost | Time | Unlocks |
|-----------|---------------|------|------|---------|
| Advanced Weapons | None | 10,000 | 1 hour | Tier 5 weapon components |
| Advanced Shields | None | 10,000 | 1 hour | Tier 5 shield components |
| Capital Class Ships | Adv. Weapons + Adv. Shields | 50,000 | 2 hours | Battlecruiser hull |
| Deep Scanning | None | 5,000 | 30 min | Enhanced scanning |
| Basic Crafting | None | 5,000 | 30 min | Crafting system |
| Basic Automation | Basic Crafting | 20,000 | 1 hour | Automation system |
| Advanced Colonies | None | 15,000 | 1 hour | Wonder construction |

Technologies with prerequisites require those techs to be completed first. Plan your research path based on your play style.
