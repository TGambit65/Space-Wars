Stats Calculation Explained

Understand how ship statistics are calculated from hull base values and component bonuses. Master these formulas to optimize your ship designs for peak performance.

Calculation System Overview

Stats Calculator System
FORMULA Final_Stat = Hull_Base + Component_Bonuses + Synergy_Bonuses
MODIFIERS Additive → Multiplicative → Percentage Bonuses
LIMITS Some stats have minimum and maximum values
Calculation Order: Base stats → Additive bonuses → Multiplicative bonuses → Percentage modifiers

Core Ship Statistics

❤️ Hull Points (HP)

Function: Determines ship durability and survival in combat

BASE Hull_Base_HP (varies by hull type)
ARMOR +25 HP per Armor Plating component
HULL_MOD +10% per Structural Reinforcement
EXAMPLE Fighter(100) + Armor(50) + 10% = 165 HP

Hull Point Modifiers

  • Armor Plating: +25 HP per component (additive)
  • Structural Reinforcement: +10% total HP (multiplicative)
  • Nanomaterial Hull: +15% total HP (multiplicative)
  • Battle Damage: Reduces maximum HP until repaired
🚀 Speed

Function: Determines travel speed and combat mobility

BASE Hull_Base_Speed (varies by hull type)
ENGINES +Engine_Rating per engine component
MASS -1 speed per 100 units of cargo mass
EXAMPLE Fighter(8) + Ion_Drive(4) - Mass(1) = 11 Speed

Speed Modifiers

  • Engine Rating: Additive bonus based on engine type
  • Cargo Mass: -1 speed per 100 units (dynamic)
  • Thruster Boosters: +20% speed (multiplicative)
  • Damage: Engine damage reduces speed proportionally
🎯 Maneuverability

Function: Affects evasion chance and weapon accuracy

BASE Hull_Base_Maneuverability
THRUSTERS +Thruster_Rating per maneuvering thruster
MASS_PENALTY -0.5 per 100 units of total mass
EXAMPLE Fighter(9) + Thrusters(2) - Mass(0.5) = 10.5

Maneuverability Factors

  • Maneuvering Thrusters: Direct additive bonus
  • Total Mass: Includes hull, components, and cargo
  • Gyroscopic Stabilizers: +15% maneuverability
  • Pilot Skill: Can provide up to +20% bonus
⚡ Power Generation & Consumption

Function: Power balance determines which systems can operate simultaneously

GENERATION Sum of all power system outputs
CONSUMPTION Sum of all component power requirements
BALANCE Generation - Consumption = Available Power
WARNING Negative balance disables components!

Power System Rules

  • Priority Systems: Life support and engines always have power
  • Optional Systems: Weapons and utilities shut down first
  • Power Efficiency: Some components reduce power consumption
  • Emergency Power: Battery backup for critical systems
📦 Cargo Capacity

Function: Maximum storage space for commodities and resources

BASE Hull_Base_Cargo (varies significantly by hull)
CARGO_BAYS +Bay_Capacity per cargo bay component
EFFICIENCY Cargo management systems +10% total
EXAMPLE Freighter(200) + Bays(180) + 10% = 418 units

Cargo Considerations

  • Specialized Bays: Some cargo requires specific storage
  • Mass Impact: Full cargo reduces speed and maneuverability
  • Access Time: Larger cargo holds take longer to load/unload
  • Security: Secure cargo bays protect valuable goods

Combat Statistics

Weapon Stats

  • Damage: Base weapon damage + targeting bonuses
  • Accuracy: Base accuracy + computer bonuses
  • Rate of Fire: Shots per combat round
  • Range: Effective engagement distance

Defense Stats

  • Shield Strength: Sum of all shield generators
  • Armor Rating: Physical damage reduction
  • Evasion: Based on maneuverability and pilot skill
  • Regeneration: Shield/hull repair rates

Optimization Techniques

Min-Maxing: Focus on maximizing key stats while accepting weaknesses in others.

Stat Prioritization by Role

COMBAT Damage > Accuracy > Defense > Speed
TRADER Cargo > Speed > Defense > Sensors
EXPLORER Speed > Sensors > Range > Versatility
FLEET Role_Primary > Communication > Coordination

Common Optimization Patterns

  • Glass Cannon: Maximum damage, minimal defense
  • Tank Build: Maximum survivability, moderate damage
  • Speed Demon: Maximum speed and evasion
  • Balanced Build: Good performance in all areas
  • Specialist: Extreme focus on one specific role
